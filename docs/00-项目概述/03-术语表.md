# 03 术语与缩略语

| 术语 | 全称 / 说明 |
| --- | --- |
| ODS | Operational Data Store，贴源层/原始数据层，保持源系统结构 |
| DWD | Data Warehouse Detail，明细数据层，清洗规范化后的事实明细 |
| DIM | Dimension Layer，维度层，全局一致性维度（师生/部门/时间等） |
| DWS | Data Warehouse Summary，汇总层，按主题预聚合的宽表 |
| ADS | Application Data Store，应用数据层，面向具体场景/大屏/API |
| DM | Data Mart，数据集市 |
| MDM | Master Data Management，主数据管理 |
| CDC | Change Data Capture，变更数据捕获 |
| ETL / ELT | Extract-Transform-Load / Extract-Load-Transform |
| SCD | Slowly Changing Dimension，缓慢变化维；SCD1 覆盖、SCD2 拉链、SCD3 新增列 |
| WAL | Write-Ahead Log（PG 的预写日志） |
| TPS / QPS | Transactions / Queries Per Second |
| RTO / RPO | Recovery Time / Point Objective |
| SSO | Single Sign-On，单点登录 |
| CAS | Central Authentication Service |
| LDAP | Lightweight Directory Access Protocol |
| RBAC / ABAC | Role / Attribute Based Access Control |
| DLP | Data Loss Prevention，数据泄露防护 |
| KMS | Key Management Service |
| HSM | Hardware Security Module，硬件加密机 |
| 等保三级 | 网络安全等级保护第三级，S3A3G3 |
| 国密算法 | 国家密码管理局发布的 SM 系列算法（SM2/3/4/9） |
| BPMN | Business Process Model and Notation 2.0 |
| SPI | Service Provider Interface，Java 服务提供接口 |
| SDK | Software Development Kit |
| SLA | Service Level Agreement |
| SLO / SLI | Service Level Objective / Indicator |
| WBS | Work Breakdown Structure，工作分解结构 |
| DDL / DML | Data Definition / Manipulation Language |
| P95 / P99 | 95/99 分位延迟 |
| TLCP | GM/T 0024 国密传输层密码协议 |
| MQ | Message Queue |
| SIS | 学校信息系统（教务/学工/财务/OA/人事/一卡通等的泛指） |
