# 06 数据集市模块详设

## 1. 模块职责

- 面向**普通用户（师生 + 部门业务员）**的"数据超市"；
- 支持浏览、搜索、收藏、订阅、申请、审批、下载、使用反馈全流程；
- 资源类型：**表、文件、数据集、API**。

## 2. 用户视角功能结构

```
数据集市 (market.bdc.edu.cn)
├── 首页 (推荐 / 热门 / 新上架 / 分类入口)
├── 分类浏览 (按主题域/部门/级别/类型)
├── 全文搜索 (关键词+分词+拼音)
├── 资源详情页 (元数据/字段/样本/质量/评分)
├── 购物车 & 申请单
├── 我的申请 (进行中/已通过/已拒绝)
├── 我的订阅 (实时数据订阅)
├── 我的收藏
├── 我的反馈 & 评分
└── 使用记录 (已获授权资源、到期提醒)
```

## 3. 资源上架

### 3.1 上架对象
- 表（直接基于仓库 DWD/DWS/ADS 的表）；
- 数据集（表 + 过滤条件 + 字段选择 + 脱敏后）；
- 文件（MinIO 中的文件集合）；
- API（由数据服务模块发布的 API）。

### 3.2 上架流程
```
1. 资源所有者（部门数据管理员）提交上架申请
2. 填写元数据：名称、描述、业务域、字段说明、样本、分级、可见范围、下载格式、使用场景
3. 数据管理员初审（字段完整性、口径）
4. 安全管理员复审（分级、脱敏、合规）
5. 上架成功 → 集市展示
```

### 3.3 资源状态
- 草稿 / 待审核 / 已上架 / 已下架 / 已过期；
- 版本号；
- 有效期（可设定）；
- 统计：浏览量、订阅量、下载量、评分。

## 4. 申请 - 审批 - 授权 - 下载全流程

### 4.1 业务流程

```
用户浏览 → 加入购物车 → 填写申请单（用途、期限、来源IP）
→ 提交 Flowable 流程 → 审批（动态路由 1~3 级）
→ 通过 → 写授权表（bdc_uac.t_data_grant）
→ 用户获得"资源访问票据"
→ 下载 / 调用 / 订阅
→ 到期/撤销 → 自动回收
```

### 4.2 申请单字段

| 字段 | 说明 |
| --- | --- |
| 申请人 | 当前登录用户 |
| 申请部门 | 自动带出 |
| 资源清单 | 可勾选多个资源（批量） |
| 字段范围 | 勾选字段子集（可达字段级） |
| 数据范围 | 行级筛选条件（如"仅本学院学生"） |
| 使用场景 | 业务文字 + 用途类别 |
| 使用期限 | 30 / 90 / 180 / 365 天 |
| 来源 IP | 可选，限制来源 |
| 附件 | 业务佐证（可选） |
| 敏感承诺书 | L3/L4 必须勾选 |

### 4.3 审批规则（详见 `06-安全与合规/04-审批流程设计.md`）
- L1 公开：自动通过 + 审计；
- L2 内部：资源部门管理员 1 级；
- L3 敏感：部门管理员 → 数据管理员；
- L4 机密：部门管理员 → 数据管理员 → 安全管理员 + 双人复核；
- 跨部门申请：升一级；
- 批量：按批次内最高级别定级；
- 超时：24/48/72 自动催办。

### 4.4 授权执行

- 对"表/数据集"：授权是行列过滤规则写入数据权限服务（`bdc_uac.t_row_rule` / `t_col_rule`）；
- 对"文件"：生成受限下载 Ticket（短时 URL + 绑定 user + IP）；
- 对"API"：写应用-API 授权表，同时生成 AK/SK（个人申请生成"个人 Token"）；
- 授权到期自动回收；到期前 7 天提醒续期。

## 5. 下载

- 所有下载集中走 `bdc-file` 服务；
- 敏感/机密文件：
  - 服务端二次解密 + 嵌入**明水印**（用户姓名/工号/时间/来源 IP）+ **隐水印**（LSB / 频域；PDF/Word 可用隐藏文字）；
  - 审计（写入不可篡改日志）；
  - 限速下载（默认 10 MB/s）；
- 下载包：ZIP 内含 README（说明脱敏规则、使用约束）+ 数据文件 + Sidecar SM3 Hash；
- 下载次数限制：默认每申请 3 次；
- 支持"链接分享给自己"但禁止转发；外发检测：若用户上传到网盘，由 DLP 旁路监控（可选组件）。

## 6. 订阅

- 实时订阅：
  - 用户订阅 API 或主题（Kafka 消费组）；
  - 平台分配独立的 Token 和 Topic 权限；
  - 消费审计 + 流量监控；
- 离线订阅：
  - 定期推送（每日/每周）到用户指定的 MinIO 目录或邮箱；
  - 由 bdc-push 执行。

## 7. 反馈与评分

- 5 星评分；
- 评论（管理员审核后公开）；
- 质量问题反馈 → 转工单到 bdc-govern 处理；
- 数据可用性打分列入资源 Owner 考核。

## 8. 数据安全强化

- 全程水印；
- 禁止客户端保存密码/明文；
- 大文件限速；
- 敏感文件在线预览模式（流式、禁止下载）；
- 打开"只读模式"配合 Office Online（可选，二期）。

## 9. 页面结构（前端 Vue）

- 首页：Banner + 推荐卡片 + 分类瓦片 + 搜索条；
- 资源详情：左侧基本信息；右侧侧栏：字段说明、样本 100 行、质量分、血缘摘要；
- 申请购物车：弹窗 + 批量；
- 我的空间：申请、订阅、授权、下载历史、反馈；
- 管理后台（/admin）：上架、审批、统计。

## 10. 接口（摘要）

```
GET  /api/market/v1/catalog/tree
GET  /api/market/v1/resource?keyword=...&category=...&level=...
GET  /api/market/v1/resource/{id}
POST /api/market/v1/resource/{id}/subscribe
POST /api/market/v1/cart/add
POST /api/market/v1/application      # 提交申请
GET  /api/market/v1/application/mine
POST /api/market/v1/application/{id}/withdraw
GET  /api/market/v1/grant/mine
POST /api/market/v1/download/presign  # 生成受控下载URL
POST /api/market/v1/feedback
```
