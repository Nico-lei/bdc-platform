# 08 数据可视化模块详设

## 1. 模块职责

- 提供**数据驾驶舱**（校领导 / 部门 / 主题）+ **BI 自助分析** + **基于开源工具的大屏设计器**；
- 支持**拖拽式搭建、组件丰富、模板 / 组件导入导出、实时渲染**；
- 满足 100 并发大屏 + 100 用户 BI 使用。

## 2. 产品结构

```
可视化平台
├── 设计器 (designer)
│   ├── 大屏 (screen)  —— 1920×1080 / 2560×1440 / 4K
│   ├── 仪表盘 (dashboard) —— 自适应
│   └── 报表 (report) —— 表格为主，二期
├── 组件市场 (widget marketplace)
├── 模板市场 (template marketplace)
├── 数据集中心 (dataset)
│   ├── SQL 数据集
│   ├── API 数据集 (接入数据服务API)
│   └── 指标数据集 (接入指标中心)
├── 主题 / 皮肤
└── 权限 (协作/公开/私有)
```

## 4. 大屏设计器（开源适配方案）

### 4.0 选型结论

- 一期不自研大屏设计器内核，采用 **GoView** 作为现成开源大屏设计器；
- 平台仅保留与高校场景强相关的轻量适配：统一登录、权限控制、数据集接入、模板管理、发布审计；
- 选型理由：与现有 **Vue 3 + TypeScript + Vite + ECharts** 技术栈兼容，前端可直接复用现有组件能力，后端仅需补齐适配接口，无需维护高复杂度自研画布内核。

### 4.1 画布能力

- 画布尺寸：预设 + 自定义；
- 网格对齐、磁吸、标尺、参考线、多选、分组、层级；
- 背景：纯色 / 图片 / GLSL 背景；
- 动画：入场 / 循环 / 退出；
- 事件：点击 / 悬停 → 跳转、联动、弹窗；
- 版本：历史版本 + 回滚。

### 4.2 组件分类

| 类 | 组件 |
| --- | --- |
| 图表 | 折线、柱状（含堆叠）、条形、饼、环、散点、气泡、漏斗、雷达、桑基、热力、树图、地图（GeoJSON）、仪表盘、双轴、词云 |
| 表格 | 基础表、滚动表、透视表、斑马条 |
| 文字 | 标题、跑马灯、数字翻牌、富文本 |
| 媒体 | 图片、视频、GIF、SVG、Iframe |
| 装饰 | 边框、线条、图形、粒子、光效 |
| 时间 | 时钟、倒计时、日历、时间轴 |
| 交互 | Tab、下拉、搜索框、按钮、日期选择 |
| 定制 | 师生分布地图、校历、成绩热力、驾驶舱 KPI 卡 |

合计预置 **40+** 组件，一期先交付 **25 个** 核心。

### 4.3 页面协议（沿用 GoView，平台做兼容映射）

页面/组件定义沿用 GoView JSON 结构，平台侧补充数据集引用、权限标签与发布元数据：
```json
{
  "type": "chart.line",
  "version": "1.0.0",
  "attr": {"w": 640, "h": 360, "x": 0, "y": 0, "z": 1, "rotate": 0, "animation": {}},
  "option": { /* ECharts 原生 option */ },
  "dataSource": {"type": "api|sql|indicator|static", "ref": "...", "params": {}, "refreshInterval": 60, "transform": "..."},
  "events": [{"event":"click","action":"jump","target":"/page/xx"}],
  "ext": {"datasetRef": "ds_xxx", "securityLevel": "L2", "publishMode": "public|intranet"}
}
```

- 优先保证与 GoView 原生结构兼容，避免维护自定义 DSL；
- 平台升级时只处理 `ext` 扩展字段，降低版本迁移成本。

### 4.4 数据集

| 类型 | 说明 |
| --- | --- |
| SQL 数据集 | 选库 + SQL 模板 + 参数 |
| API 数据集 | 选取数据服务 API + 参数绑定 |
| 指标数据集 | 选指标 + 维度筛选 |
| 静态数据集 | 手动维护 Mock（演示用） |

数据集缓存：默认 60s；支持手动刷新 + 事件驱动刷新；热更新。

### 4.5 运行态

- 独立播放器（Iframe 嵌入/ 全屏）；
- 支持定时自动刷新；
- 断网自动降级（最后一次缓存）；
- 支持大屏拼接（多屏幕拖拽排布）。
- 设计器以开源前端应用方式集成，可采用 iframe 或微前端挂载到 `bdc-visual` 管理台。

### 4.6 模板与组件导入导出

- 模板：整屏打包 ZIP（`manifest.json` + `pages/` + `datasets/` + `assets/`）；
- 组件：单组件 JSON + 预览图；
- 导入前安全扫描（JS 表达式 / 外链资源）；
- 市场分类：驾驶舱 / 年度总结 / 招生 / 就业 / 安全 / 资产 …
- 模板优先复用 GoView 社区模板，再做校内主题化封装，减少自制模板工作量。

## 5. BI 自助分析

### 5.1 操作能力

- 选表/指标 → 拖维度/度量 → 选图表 → 过滤/排序/分组 → 保存；
- 图表联动、钻取；
- 导出 CSV / PNG / PDF；
- 嵌入（iframe） / 分享（短链 + 授权）；

### 5.2 权限

- 数据权限继承平台权限体系；
- 报表权限：私有 / 团队 / 共享（角色）；
- 发布到门户需二级审批。

## 6. 数据驾驶舱（预置）

- **校领导驾驶舱**：学生总览、教学运行、科研实力、财务概览、安全态势；
- **部门驾驶舱**：学工、教务、财务、后勤、人事、图书 等；
- **主题驾驶舱**：招生、就业、本科教学评估、研究生培养质量、突发事件应急响应（通用模板，覆盖公共卫生 / 安全事件 / 自然灾害等场景）。

每个驾驶舱按"KPI 卡 + 趋势 + 分布 + TopN + 告警 + 地图"结构设计，预置模板可改。

## 7. 性能

| 项 | 目标 |
| --- | --- |
| 首屏 | ≤ 3s |
| 单组件刷新 | ≤ 1s（简单）/ ≤ 3s（复杂） |
| 100 并发大屏 | 平稳渲染 |
| 单页组件数 | ≤ 50 |
| 数据量 | 单组件返回 ≤ 10 万行（超过使用 Top N + 分页） |

优化：
- 数据集聚合到 ADS；
- Redis 缓存 60s；
- 前端虚拟滚动、按需加载、Worker 解析。

## 8. 安全

- 页面水印；
- 敏感字段走数据服务动态脱敏；
- 分享链接使用带签名的短链（过期、次数限制）；
- 仅 L1/L2 级资源允许在公开大屏展示；L3+ 需审批并限内网播放；
- 运行态禁止在控制台泄露 SQL。

## 9. 国际化 / 可访问性

- 中文 + 英文 + 繁体（UI 文案）；
- 色盲友好配色预置；
- 屏幕阅读器（Aria 标签基础支持）；
- 键盘可达性。
