# 02 接口清单（核心 API 目录）

> 清单含 **管理面**（给前端管理台 / 后台 / 运营用）与 **服务面**（给业务系统 / 应用调用）两类，按微服务领域分组。
> 每条接口含：Method、Path、功能简述、鉴权、限流级别。详细请求 / 响应结构在各服务 OpenAPI yaml 中维护（随代码交付）。
