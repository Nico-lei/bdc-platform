# portal

| 项 | 值 |
| --- | --- |
| 工程 | `bdc-frontend/portal` |
| 用途 | 统一门户首页 + 师生入口 |
| dev 端口 | `5173` |
| 落地阶段 | **P2.5**（当前 P0.1.2 仅骨架） |

## 启动

```bash
cd bdc-frontend
pnpm install
pnpm dev:portal
# 或
pnpm --filter portal dev
```
