# designer

| 项 | 值 |
| --- | --- |
| 工程 | `bdc-frontend/designer` |
| 用途 | 大屏 / ETL / 表单 设计器 |
| dev 端口 | `5177` |
| 落地阶段 | **P3.3 / P4.8**（当前 P0.1.2 仅骨架） |

## 启动

```bash
cd bdc-frontend
pnpm install
pnpm dev:designer
# 或
pnpm --filter designer dev
```
