# screen

| 项 | 值 |
| --- | --- |
| 工程 | `bdc-frontend/screen` |
| 用途 | 数据大屏运行态 |
| dev 端口 | `5175` |
| 落地阶段 | **P4.8**（当前 P0.1.2 仅骨架） |

## 启动

```bash
cd bdc-frontend
pnpm install
pnpm dev:screen
# 或
pnpm --filter screen dev
```
