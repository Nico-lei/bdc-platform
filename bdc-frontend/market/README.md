# market

| 项 | 值 |
| --- | --- |
| 工程 | `bdc-frontend/market` |
| 用途 | 数据集市门户（用户视角） |
| dev 端口 | `5176` |
| 落地阶段 | **P4.7**（当前 P0.1.2 仅骨架） |

## 启动

```bash
cd bdc-frontend
pnpm install
pnpm dev:market
# 或
pnpm --filter market dev
```
